# ConfigMap : Rule of Thumb
    ConfigMap Store:
    API URLs
    Ports
    Feature Flags
    Environment Names
    Hostnames
Example:
    DB_HOST
    PORT
    NODE_ENV
    FRONTEND_URL

# Secrets:
    Store:
    Passwords
    JWT Secrets
    API Keys
    OAuth Secrets
    Database Credentials
    Certificates

Example:
    JWT_SECRET
    DB_PASSWORD
    AWS_ACCESS_KEY
    AWS_SECRET_KEY
    GOOGLE_CLIENT_SECRET
